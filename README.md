ERFlute
=======================
ERD Modeling Tool for Startup & Incremental Development as Eclipse Plugin

## Features

- Simple but Graphical
- Simple but Functional
- Mergeable ERD

## Install ERFlute

```
dropins
 |-ERFlute
    |-eclipse
       |-plugins
          |-org.dbflute.erflute_0.5.8.jar
```
